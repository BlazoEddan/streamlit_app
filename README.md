# Real-Time Object Detection and Tracking using AI and Webcam

Live Object Detection & Tracking using Streamlit and YOLOv8
📌 Project Overview

This project is a real-time Artificial Intelligence web application built using Python, Streamlit, and the YOLOv8 object detection model.
The application captures live video from a webcam and performs object detection and tracking directly on the video stream.

Detected objects such as people, cell phones, bottles, and other everyday items are highlighted using bounding boxes and labeled in real time.

This activity demonstrates how Computer Vision and Artificial Intelligence can be integrated into a web-based application for real-world use cases.

📂 Features
✅ Live Webcam Feed
Displays the webcam stream directly in the browser.
✅ Real-Time Object Detection
Detects common objects using YOLOv8.
✅ Bounding Boxes & Labels
Shows detected objects with labels and confidence scores.
✅ Object Tracking
Tracks moving objects smoothly across multiple frames.
✅ User-Friendly Interface
Simple and interactive Streamlit web interface.
